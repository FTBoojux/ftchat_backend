## FTChat

这是一个用于个人学习使用的聊天应用`FTChat`对应的后端程序，当前正在逐步设计和开发中。

本项目基于Django框架设计和开发。

对应的前端程序：[FTChat](https://github.com/FTBoojux/ftchat)

对应的实时聊天模块：[FTChat-channel](https://github.com/FTBoojux/ftchat-channel)
